How to Work Here

Rule: Only edit inside your own person section below, between the BEGIN/END markers. This minimizes merge conflicts.

  1. Each person has identical TODOs: Article, Lists, Table, Form, Back-to-top link.
  2. Keep IDs unique (pre-filled per person: e.g., bartek-articles, lukasz-articles).
  3. Commit often. Create your own feature branch (e.g., feat/bartek-todos or feat/lukasz-todos).
