Cheat Sheet (fast recap)

Specificity

  • Inline > #id > .class / [attr] / :pseudo > element
  • Later rules win if specificity ties
  • Avoid !important in normal code

Box Model

  • box-sizing: border-box; keeps width predictable
  • Margin collapses vertically between blocks

Layout

  • Flex: one-dimensional, main/cross axis
  • Grid: two-dimensional, rows × columns

Units

  • rem = root font size, em = parent
  • vw/vh = viewport, clamp(min, pref, max)

Tip: To "toggle dark mode" for testing, temporarily add bartek-dark to the #bartek element (via DevTools) or lukasz-dark to #lukasz.
